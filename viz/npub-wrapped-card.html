Take the winning W section, wrap it into a square grid. Compact, memorable, easy to verify.

The Problem with 1D Strings

A 16-char winning section as a string:
q p q p q p q p q p q p q p q p

Same 16 chars wrapped into 4×4:
1D string: hard to verify at a glance
qpqpqpqpqpqpqpqp
→
2D grid: instantly recognizable shape
The 2D grid takes less space, creates a visual SHAPE, and is much easier to compare. You don't read each character — you recognize the pattern of colors.

How Wrapping Works

Grid side = √W rounded up. Characters fill left-to-right, top-to-bottom.
W=4 → 2×2 grid
W=9 → 3×3 grid
W=16 → 4×4 grid
W=25 → 5×5 grid
Short W (2×2, 3×3) = tiny grid, very simple shape, lower z-score.
Long W (4×4, 5×5) = larger grid, more complex shape, higher z-score.
The grid SIZE encodes the fingerprint strength. Bigger grid = more work invested.
